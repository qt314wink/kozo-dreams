# Kozo Dreams — Dark Washi Fantasy Design System

> A design system where traditional Japanese papercraft meets dark whimsical fantasy. Twenty unique components. Five sacred dyes. One living interface.

## Quick Start

Open `index.html` in any browser. Zero dependencies. Zero build step.

```bash
git clone https://github.com/qt314wink/kozo-dreams.git
cd kozo-dreams
open index.html
```

## Features

- **Zero Dependencies** — Pure vanilla HTML/CSS/JS
- **Dark Mode** — Automatic + manual toggle with smooth transitions
- **Responsive Atmosphere** — Adapts fidelity based on device capability
- **Scroll Animations** — Intersection Observer-driven reveals
- **Copy-to-Clipboard Tokens** — Click any color token to copy
- **Living Mascot** — Interactive fantasy creature (click 5x for easter egg)
